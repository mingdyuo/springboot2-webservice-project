# springboot2-webservice-project

### 프로젝트 정보

1. Java 11
2. Spring Boot 2.1.9
3. Build tool : Gradle
4. Front-end Template : Mustache
5. DB : MySQL
6. DB manage : AWS RDS
7. Deploy : AWS EC2
8. version manage : git
